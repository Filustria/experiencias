A Plain Markdown Post

Dec 30, 2016

This is a post written in plain Markdown (*.md) instead of R Markdown (*.Rmd). The major differences are:

  1. You cannot run any R code in a plain Markdown document, whereas in an R Markdown document, you can embed R code chunks (```{r});
  2. A plain Markdown post is rendered through Blackfriday, and an R Markdown document is compiled by rmarkdown and Pandoc.

There are many differences in syntax between Blackfriday’s Markdown and Pandoc’s Markdown. For example, you can write a task list with Blackfriday but not with Pandoc:

Similarly, Blackfriday does not support LaTeX math and Pandoc does. I have added the MathJax support to this theme (hugo-lithium-theme) but there is a caveat for plain Markdown posts: you have to include math expressions in a pair of backticks (inline: `$ $`; display style: `$$ $$`), e.g., $S_n = \sum_{i=1}^n X_i$.1 For R Markdown posts, you do not need the backticks, because Pandoc can identify and process math expressions.

When creating a new post, you have to decide whether the post format is Markdown or R Markdown, and this can be done via the ext argument of the function blogdown::new_post(), e.g.

blogdown::new_post("Post Title", ext = '.Rmd')
  1. This is because we have to protect the math expressions from being interpreted as Markdown. You may not need the backticks if your math expression does not contain any special Markdown syntax such as underscores or asterisks, but it is always a safer choice to use backticks. Brainstorm de 07.03.2018

Trabalharemos com o objetivo de construir oito atividades acadêmicas voltadas para os nossos R2. As atividades dos R1 serão aproveitamento das atividades dos R2.

O que é raciocínio clínico. Modelos de Raciocínio Clínico. O que é MBE. Ferramentas de busca de informação. Nomograma de Fagam. Manejando a incerteza. Conceitos de Epidemiologia clínica (Medidas de frequência - prevalência, incidência, probabilidades. Epi clínica para diagnóstico - sensibilidade, espec, LR, Epi clínica para terapêutica - medidas de efeito - RR, RA, RRR, RAR, NNT). Erros e viéses de raciocínio. Avaliação crítica de estudos. Capacidade de discernir informações médicas (validade, magnitude, impacto). Saber avaliar estudos. Desfecho. Prognóstico. Rastreamento. Lista de problemas do paciente. Organização de dados para o raciocínio clínico. Interface do raciocínio clínico com a comunicação clínica Ergometria. O que o paciente pode ter e o que ele não deve ter Estratégias de raciocínio. Como ensinar raciocínio clínico no ambulatório. Como construir o conteúdo de forma a facilitar o raciocínio dentro de um livro ou artigo. Grau de recomendação e níveis de evidência.
